# Audit Rules

Every capability should answer these questions:

1. Does the engine produce it?

2. Is it persisted in runtime?

3. Is it benchmarked?

4. Is it projected?

5. Is it visible in the executive experience?

6. Should it be visible?

7. Does it improve executive decision making?

8. Should it participate in the simulated organization pressure test?
