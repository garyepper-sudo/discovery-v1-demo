# Living Understanding Organism

## Philosophy

The organism is NOT a visualization of evidence.

The organism is a visualization of understanding.

Evidence creates themes.

Themes create mechanisms.

Mechanisms support beliefs.

Beliefs form understanding.

Understanding evolves over time.

## Rendering Architecture

Layout

↓

Render State

↓

Rendering

Rendering only draws.

This separation supports

- reasoning highlighting

- search

- timeline

- delta

- enterprise organism

without changing rendering logic.
